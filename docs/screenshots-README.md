## How to Take Screenshots

### On Windows:
1. Press `Windows + Shift + S` for Snipping Tool
2. Select the area you want to capture
3. Save with the recommended filename

### On Mac:
1. Press `Shift + Command + 4`
2. Select the area you want to capture
3. Save with the recommended filename

### Using Browser Developer Tools:
1. Press `F12` to open Developer Tools
2. Toggle device toolbar (responsive mode)
3. Take screenshot of different sections
4. Use browser's screenshot feature if available

## Tips
- Use full-screen screenshots for best quality
- Ensure all text is readable
- Capture important interactive elements
- Show hover states if possible
- Include both desktop and mobile views
