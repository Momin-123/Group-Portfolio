## ğŸš€ How to Use

1. **Open the Portfolio:**
   - Simply open `index.html` in your web browser
   - Or use a local server (recommended)

2. **Add Your Team Photos:**
   - Place 5 profile pictures in the `assets/` folder
   - Recommended size: 500x500px (square)
   - Update image paths in `index.html`

3. **Add Project Images:**
   - Place project screenshots in the `assets/` folder
   - Recommended size: 1200x800px (landscape)
   - Update image paths in the projects section

4. **Customize Content:**
   - Edit team member information in `index.html`
   - Update skills in the skills section
   - Add your actual projects
   - Contact information is already updated
