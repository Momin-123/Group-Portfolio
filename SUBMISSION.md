## How to View

1. Open `index.html` in any modern web browser
2. Or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (with http-server)
   npx http-server
   ```
3. Navigate to `http://localhost:8000`
